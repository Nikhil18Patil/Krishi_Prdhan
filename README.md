# Krishi_Prdhan
Kirshi Pradhan is a website that was created to help farmers increase their awareness and productivity. 
The website provides farmers with market information, weather reports, crop information, trends, and more. The website was built using Django, a popular web framework. 
Django is known for its ease of use and its ability to scale to large websites. 
The website also uses REST APIs to allow farmers to access the information they need on their mobile devices.
